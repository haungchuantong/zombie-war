# Generated Pixel Art Assets

The game can run without these files because `main.js` includes Canvas fallbacks. Add AI-generated PNGs here using the exact names below to upgrade the visuals.

## Style Prompt Base

Top-down / slight isometric pixel art for a dark zombie survival game, crisp 32-bit pixel edges, readable silhouette, dark green and rust survival palette, transparent or flat chroma-key background, no text, no watermark, no UI frame.

## Expected Files

- `weapon-pistol.png`
- `weapon-smg.png`
- `weapon-assault-rifle.png`
- `weapon-shotgun.png`
- `weapon-sniper-rifle.png`
- `weapon-flamethrower.png`
- `weapon-tesla-gun.png`
- `weapon-grenade-launcher.png`
- `weapon-chest.png`
- `reward-chest.png`
- `burrow-warning-crack.png`
- `hunter-zombie.png`

Keep weapon icons around 64x64 or 96x96 PNG. Keep effect props centered with padding so the Canvas can scale them cleanly.
